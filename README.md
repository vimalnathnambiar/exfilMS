A command line interface (CLI) application to extract and filter MS data.

## Installation

```bash
$ cd exfilms
$ sudo npm install
$ sudo npm install -g .
```

<!-- `$ npm i exfilms` -->

## Usage

```bash
$ exfilms --inputDir (or -i) "/path/to/input/directory/"

# Use interactive mode
$ exfilms --interactive

# Show help
$ exfilms --help
```
