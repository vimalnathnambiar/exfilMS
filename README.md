A command line interface (CLI) application to extract and filter MS data.

## Installation

```bash
$ cd exfil-ms
$ sudo npm install
$ sudo npm install -g .
```

<!-- `$ npm i exfil-ms` -->

## Usage

```bash
$ exfil-ms --inputDir (or -i) "/path/to/input/directory/"

# Use interactive mode
$ exfil-ms --interactive

# Show help
$ exfil-ms --help
```
